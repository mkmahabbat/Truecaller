NumberInfoBot - Telegram Bot to Check Mobile Number Information

HOW TO USE:
1. Replace 'YOUR_TELEGRAM_BOT_TOKEN' in main.py with your actual bot token from BotFather.
2. Install dependencies:
   pip install -r requirements.txt
3. Run the bot:
   python main.py

Then send any number with a country code (e.g., +919999999999) to the bot in Telegram.

It will reply with country, carrier, and more.
